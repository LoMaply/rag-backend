IRIS backend + db for RAG chatbot. Requires InterSystems IRIS. Used in tandem with [rag-frontend](https://github.com/LoMaply/rag-frontend).

Before running, use irispip to install the following:

- langchain
- langchain-cohere
- langchain-community
- langchain-core
- langchain-iris
- pypdf
- unstructured
- sqlalchemy-iris
